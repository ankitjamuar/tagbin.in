Tweetcafe Details

  • How it works
  • Features
  • Specifications
  • Shelf like installation with a number of lockers and a display screen with instructions.
  • Gifts are kept inside the lockers.
  • User tweets with the provided hashtag of the event/brand.
  • The tweet is detected and a digital fortune wheel is spinned on the screen to choose the lucky number for the user.
  • The lucky number is used to open the specific locker and the user picks up the gift and gets a memorable experience.
  • Electronically controlled 24 gift chambers (size variable)
  • Large format screen with twitter and FB #tag integration possible
  • Customized vinyl branding options
  • Variable number of lockers
  • 1 LCD/LED screen (variable size)
  • i5, 4GB RAM, 120GB HDD
